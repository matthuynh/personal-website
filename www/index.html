Shrinkly - A URL shortener
JavaScript, MongoDB, Express.js, ReactJS, Node.js, Bulma

Worked with a partner to develop a full-stack URL shortener service that allows a user to input long URLs to create their own custom short URL. The service is very similar to other services such as bit.ly and tinyurl.cc. The application was built with the MERN JavaScript stack (MongoDB, Express.js, ReactJS, Node.js). The application is styled with Bulma.
The application is currently being hosted on Heroku, and can be visited at this link! Note that the application may take a few moments to initially load up as we are currently using Heroku's free student plan.

UofT Shuttle Bus Scheduler
JavaScript, MongoDB, Express.js, Pug.js, Node.js, Python

Developed a full-stack shuttle bus scheduler web application that allows a user to provide their current location and a desired departure time to plan out a UofT Shuttle Bus trip. Used Node.js with Express.js to provide logic and handle endpoints, as well as implement connectivity with MongoDB to retrieve bus time data. Utilized the Google Directions REST API in order to retrieve trip data such as estimated walk. Wrote a Python script that retrieves data from a UofT website and stores it into a Mongo database.

Financial Stock Screener
Java, JavaFX, SQLite

This is a Java desktop application that is capable of displaying time series data and technical indicators of a stock. The GUI was built with JavaFX, and the AlphaVantage API was used to retrieve stock data, which is then categorized efficiently and stored in a local SQLite database.
